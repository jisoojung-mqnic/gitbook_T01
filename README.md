# 😆 깃북 테스트

Cloud Radar를 활용하면 클라우드 비용을 분석하여 자원을 효율적으로 관리할 수 있습니다. 클라우드 자원 사용량과 비용에 대한 상세한 정보를 제공하여, 최적화된 결정을 내릴 수 있도록 도와드립니다. 클라우드 비용을 최소화하고 자원을 효율적으로 활용하여 비즈니스 목표를 달성할 수 있도록 정보를 제공합니다.



### 브라우저 설정

{% hint style="info" %}
Cloud Radar는 Chrome 브라우저에 최적화 되어 있어 Chrome 사용을 권장 드립니다.&#x20;

Chrome 브라우저 기준으로 설정 방법을 안내 드립니다.
{% endhint %}

Cloud Radar는 로그인에 필요한 세션 키를 쿠키에 저장하기 때문에 브라우저 설정에서 **서드 파티 쿠키 허용**이 필요합니다.<br>

1. 우측 상단 제어버튼을 클릭하여 설정으로 이동합니다.

<figure><img src=".gitbook/assets/titan 연계현황_240419 (1).png" alt=""><figcaption></figcaption></figure>

2. 개인 정보 보호 및 보안 탭으로 이동하여 **서드 파티 쿠키** 탭을 클릭합니다.
3. "**서드 파티 쿠키 허용**" 으로 설정을 변경합니다.



### MFA(Multi-Factor Authentication) 입력

1. 먼저 [MFA 설정 가이드](/broken/pages/ZBEIpNdIdfcwR23Tp2HT)로 이동하여 MFA를 설정합니다.&#x20;
2. MFA 설정을 마치면, 다음 로그인부터는 아래와 같은 MFA 입력 화면이 나타납니다.



<figure><img src=".gitbook/assets/titan 연계현황_240419.png" alt=""><figcaption></figcaption></figure>
